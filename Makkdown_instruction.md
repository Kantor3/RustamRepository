# Инструкция для работы с Markdown

## Выделение текста
Чтобы выделить текст курсивом необходимо обрамить его звездочками (*). Например, *вот так*.

Чтобы выделить полужирным - необходимо обрамить его двойными звезочками (**). 
Пример, **вот этак**
